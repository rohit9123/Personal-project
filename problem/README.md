# CSES & Competitive Programming Problems

This directory is dedicated to solving competitive programming problems (e.g., CSES).

## Getting Started

A Java template is provided in [Template.java](file:///Users/rohit.kumar.4/Documents/interview-prep/problem/Template.java). 

### How to use:
1. Copy the contents of `Template.java` or duplicate it for a new problem (e.g., `WeirdAlgorithm.java`).
2. Update the class name in the file to match the new file name (e.g., `public class WeirdAlgorithm`).
3. Implement your logic in the `solve` method.

### Compilation and Execution:
To compile and run your Java files via terminal:

```bash
# Compile
javac WeirdAlgorithm.java

# Run with standard I/O
java WeirdAlgorithm

# Run with input redirection (recommended for testing)
java WeirdAlgorithm < input.txt
```
